   {lenta.ru:{название_статьи1: путь_к_файлу1, 
              название_статьи2: путь_к_файлу2,
              },
    ria.ru:{название_статьи1: путь_к_файлу1, 
            название_статьи2: путь_к_файлу2,
            },         
    }

2) Реализовать веб-интерфейс с поиском по ключевым словам в заголовке статей.
   - Файлы хранятся в формате html.
   - После выбора статьи в веб-интерфейсе, идет переадресация на страницу.

3) Реализовать генератор статичных страниц, который автоматически пере собирает 
   сайт при появлении новых статей.

  3) Реализовать планировщик задач, который будет опрашивать сайт по расписанию.

  3) Реализовать мониторинг директории, в которую складываются файлы со списком url.
